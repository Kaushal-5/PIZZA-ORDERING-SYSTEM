# Pizza Ordering App

Welcome to the Pizza Ordering App! This application allows users to browse menus, customize pizzas, and place orders directly from their Android device.

## Features

- **User Authentication**: Secure Login and Sign Up.
- **Menu Browsing**: Explore a variety of pizzas.
- **Customization**: Choose your toppings and sizes.
- **Cart Management**: Add items to cart and review orders.

## Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-username/pizza-ordering-app.git
   ```
2. **Open in Android Studio**: File -> Open -> Select project folder.
3. **Build**: Let Gradle sync and build the project.
4. **Run**: Select an emulator or device and run the `app` configuration.
